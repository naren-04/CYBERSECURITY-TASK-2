# CYBERSECURITY-TASK-2
# Task 2 – Phishing Email Analysis

## Objective
To identify phishing indicators in a suspicious email.

## Files
SAMPLE PHISHING EMAIL.txt – Sample phishing email  
report.txt – Analysis report  

## Tools Used
The following free online tools were used to analyze email headers:

1. Google Admin Toolbox – Messageheader  
https://toolbox.googleapps.com/apps/messageheader/

2. MXToolbox – Email Header Analyzer  
https://mxtoolbox.com/EmailHeaders.aspx  

These tools help detect:
- Real sender IP address
- Email spoofing
- SPF and DKIM status

## Findings
The phishing email contained multiple warning signs:
- Fake sender domain (email spoofing)
- Suspicious URL that is not an official website
- Urgent and threatening language
- Generic greeting
- Request to click a link and enter credentials

## Conclusion
The email shows clear signs of phishing, including spoofing, fake links, urgency, and social engineering. These techniques are commonly used by attackers to trick users into revealing sensitive information.
